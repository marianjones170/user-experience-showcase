Exercise 13: Enumeration Documentation

I documented an enumeration in the form of a JavaScript constant object for a fictitious Learning Management System for this exercise.

const Role = {
  Student: 0,
  AssistTeacher: 1,
  Teacher: 2,
  Admin: 3
};

Defines the user roles and permissions.

Syntax:

Javascript

Name Description Value
Student AssistTeacher Teacher Admin
Students can enroll and submit assignments. Assistant teachers can grade homework. Teachers can submit final grades. Administrators can perform any action, including creating and deleting courses.
